# Cognizant Digital Nurture 5.0 - Java Full Stack Engineering Journey

Welcome to my **Cognizant Digital Nurture 5.0 (JavaFSE)** learning repository.

This repository captures my hands-on progress across Java fundamentals, design patterns, testing, SQL/PL-SQL, Spring Core, Spring Boot REST services, and Spring Data JPA.

## Program Focus

This journey is designed to build end-to-end Full Stack Engineering skills through practical exercises and mini-projects:

- Backend development with Java and Spring ecosystem
- Web/API development with Spring Boot
- Database design and querying with SQL/PL-SQL
- Unit testing and mocking with JUnit and Mockito
- Software design principles and design patterns
