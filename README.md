# Cognizant Digital Nurture 5.0 - Java FSE Deep Skilling

This repository contains my hands-on solutions for the **Cognizant Digital Nurture 5.0 – Java Full Stack Engineer (FSE) Deep Skilling Program**.

It covers the full learning track across Java, Design Patterns, Data Structures & Algorithms, PL/SQL, JUnit, Mockito, Spring Framework, Spring Boot, REST APIs, Spring Security, JWT Authentication, Microservices, and ReactJS.

## Overview

The goal of this repository is to document practical exercises and implementations that build real-world full stack engineering skills across backend, database, testing, and frontend development.
