# Week 3 - Spring REST Using Spring Boot 3

## Overview

This week focuses on building REST-based applications with Spring Boot 3. The exercises progress from project setup to REST endpoints, country service logic, and JWT-based authentication.

## Projects

### 1. Create Spring Web Project

- Sets up the initial Spring Boot web application structure.
- Establishes the base project configuration for the remaining exercises.

### 2. Load Country From XML

- Demonstrates how to load country data from an XML source.
- Introduces XML-based data handling inside a Spring Boot application.

### 3. Hello World REST

- Builds a basic REST endpoint.
- Verifies that the Spring Boot web service setup is working correctly.

### 4. Country Web Service

- Implements a REST service for country-related data.
- Focuses on request handling and JSON responses.

### 5. Get Country By Code

- Adds an endpoint to retrieve country details using a country code.
- Demonstrates path variable handling in REST APIs.

### 6. JWT Authentication

- Introduces JWT-based authentication and authorization.
- Shows how to secure REST APIs in Spring Boot.

## Technologies Used

- Java 17
- Spring Boot 3
- Spring Web
- Spring Security
- Maven
- Eclipse IDE / VS Code

## Learning Outcomes

- Understand the basics of Spring Boot REST development.
- Create and organize Spring Boot web applications.
- Work with XML data inside a Spring Boot project.
- Develop REST controllers and endpoints.
- Handle path variables and JSON responses.
- Secure APIs using JWT authentication.
